Generalize Evolva Task-set Tooling

Summary

Reframed local CSV/attachment task preparation as generic taskset support so Evolva presents as a general agent harness.

What changed

  • Renamed support modules to evolva.tools.taskset and evolva.eval.taskset.
  • Replaced the old command namespace with evolva taskset health|smoke|run|inspect-file.
  • Renamed built-in tools to taskset_context, taskset_smoke_check, and taskset_tool_health.
  • Updated tests, capabilities, and MCP preset descriptions.
